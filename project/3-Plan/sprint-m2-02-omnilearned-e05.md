# Sprint M2-02 OmniLearned E0.5

**Milestone:** M2 - Data and Model Feasibility

**Goal:** Validate the pinned OmniLearned dependency, native A-D inputs, custom-data indices, checkpoint identity, backbone loading, and finite tiny fine-tuning before E1.

**Architecture:** OmniLearned remains an isolated external process. particleML owns index/checkpoint validation, allowlisted commands, input policy, evidence, and claim-aware fallback decisions.

**Tech Stack:** Python, pinned PyTorch environment, RunPod RTX 4090, official OmniLearned revision `5091595d226b6021e967ab2ecfff832f40c026f6`, HDF5.

## 1. Sprint Goal

Pass AC-E05-001 on RunPod or stop with an explicit, evidence-backed supervised fallback/claim-narrowing decision.

Core objectives:

- Build and validate official custom-data indices for A-D.
- Freeze checkpoint identity, license, corpus, normalization, and tensor-load audit.
- Demonstrate finite A-D forward/backward behavior and decreasing tiny-run loss.
